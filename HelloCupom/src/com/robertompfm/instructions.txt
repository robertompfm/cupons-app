Boa noite! Segue o miniprojeto:

Sistema de cadastro de cupons
- A tela inicial deve listar os cupons já cadastrados ordenados por data de data cadastro. Essa tela possui um botão para adicionar novo cupom
- Incluir um novo cupom. Deve-se informar:
    Código do cupom (o código informado deve composto por letras e números, pelo menos uma letra),
    descrição,
    valor,
    data de expiração,
    data de uso e
    Situação (Ativo, Expirado, Utilizado)
- Os cupons expirados devem aparecer em destaque na listagem
- Os cupons expirados não podem ser alterados
- Deve ser possível filtrar os cupons na listagem por situação
- Deve ser possível filtrar os cupons na listagem que expiram em um determinado período (data início e data fim)

Por favor, disponibilizar no Github quando concluir e enviar o link por aqui mesmo.
